# Mangati Platform (Manga/Manhwa Reader)

هذا المستودع يضع الأساس لمنصة احترافية لقراءة المانجا والمانهوا مع بنية كاملة للـ Front-end والـ Back-end وقاعدة البيانات والبنية التشغيلية.

## البنية العامة
```
apps/
  web/     Next.js 14 (App Router) + Tailwind + ShadCN
  api/     NestJS (REST + GraphQL)
```

## مميزات مخططة
- الصفحة الرئيسية (Carousel + أحدث الفصول + آخر الأعمال)
- صفحة عمل تفصيلية (غلاف، تصنيفات، قصة، تقييم، فرق ترجمة، فصول...)
- قارئ عمودي مع حفظ التقدم
- نظام حسابات متكامل (JWT + Refresh + Email Verification)
- لوحة تحكم إدارة متقدمة
- أداء عالي وتخزين سحابي (S3)

## التشغيل (مبدئياً)
- إعداد المتغيرات في `.env`
- تشغيل الخدمات عبر Docker Compose

```bash
docker compose up -d
```

> ملاحظة: هذه النسخة هي هيكل أولي قابل للتوسيع.
